Katana Script Launcher

Python | PyQt | Windows 11 | CentOS7 GitHub

During my time as a Support Engineer at Foundry, a good amount of it was spent regression testing bugs and investigating particular issues. This requires using several scripts to set enviornment variables while Katana is being launched for different renderers, USD, custom plugins, etc. This launcher allowed me to store all of these custom scripts and interact with them in an intuitive way.

This script will scan for installed versions of Katana and different renders, primarily Arnold and Renderman, including all different versions installed and allow the user to switch between them quickly.

Master Asset Pipeline

Maya | Python | PyMel GitHub

This asset focused pipeline separates every asset into its own separate folder and every department step: modeling, texturing, rigging, and lighting. The work folder allows artists to create whatever they like before committing any changes. Once they are ready to commit, their scene must first pass a series of checks I've created before another department may use that scene.
